# Supported Languages & Frameworks

## JavaScript/TypeScript
- **Linting**: ESLint with configurable rules
- **Formatting**: Prettier with consistent style
- **Testing**: Jest, Vitest, Mocha, or npm test
- **Security**: npm audit, dependency checks

## Rust
- **Linting**: Clippy with comprehensive checks
- **Formatting**: rustfmt with standard style
- **Testing**: cargo test with coverage options
- **Security**: cargo audit, dependency checks

## Python
- **Linting**: Flake8, Black, Ruff
- **Formatting**: Black, isort
- **Testing**: pytest, unittest
- **Security**: bandit, safety checks

## Go
- **Linting**: golangci-lint, go vet
- **Formatting**: gofmt, goimports
- **Testing**: go test with coverage
- **Security**: gosec

## General
- **Git Hooks**: pre-commit validation
- **CI/CD**: Pipeline integration scripts
- **Documentation**: README generation
- **Dependencies**: License compliance checks
