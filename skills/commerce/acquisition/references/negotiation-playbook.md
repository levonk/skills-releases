# Negotiation Playbook

## Universal Principles

1. **Always disclose agent status** — "I am a purchasing agent representing the buyer"
2. **Anchor low, justify with data** — First offer sets the frame; use market comps
3. **Silence is a weapon** — After making an offer, wait; let the seller respond first
4. **Bundle for concessions** — "If you include X, I can agree to Y"
5. **Walk-away point is real** — Know the max and never exceed it
6. **Time is leverage** — Sellers with aged inventory are more flexible
7. **Cash talks** — Immediate payment or fewer contingencies can justify lower price

## Category-Specific Tactics

### Vehicles (New)

| Tactic | Details |
|--------|---------|
| **Opening offer** | Invoice price minus holdback (typically 2–3% of MSRP) |
| **Research** | Check [TrueCar](https://www.truecar.com/), [Edmunds TMV](https://www.edmunds.com/), [KBB Fair Purchase Price](https://www.kbb.com/) |
| **Timing** | End of month, end of quarter, end of model year (Sept–Nov) |
| **Leverage** | Multiple dealer quotes (email dealers); manufacturer incentives/rebates |
| **Walk-away** | Dealer cost + $500 is a strong deal for most non-luxury vehicles |
| **Avoid** | Dealer add-ons (paint protection, fabric guard, VIN etching) — all are high-margin |
| **Finance** | Get pre-approved from bank/CU first; only accept dealer financing if they beat the rate |

### Vehicles (Used)

| Tactic | Details |
|--------|---------|
| **Opening offer** | 15–20% below asking (private party); 10–15% (dealer) |
| **Research** | [KBB Private Party](https://www.kbb.com/), eBay sold listings, [CarGurus](https://www.cargurus.com/) price analysis |
| **Leverage** | Pre-purchase inspection findings, needed maintenance items, comparable listings |
| **Walk-away** | Private party wholesale value + needed repairs; dealer trade-in value + 15% |
| **Key phrase** | "I have cash ready today if we can agree on $X" |

### Luxury Watches

| Tactic | Details |
|--------|---------|
| **Opening offer** | Grey market price (Jomashop, Chrono24) — typically 10–30% below AD retail |
| **Research** | [WatchCharts](https://watchcharts.com/) for market value; [Chrono24](https://www.chrono24.com/) for comparables |
| **Leverage** | Purchase history with the AD; willingness to buy complementary pieces; cash payment |
| **AD strategy** | Build relationship; ask about pieces coming off waitlist; buy entry-level first to build credit |
| **Grey market** | Always verify seller reputation, return policy, warranty card authenticity |

### Used Marketplace Items (FB Marketplace, Craigslist, OfferUp)

| Tactic | Details |
|--------|---------|
| **Opening offer** | 25–35% below asking |
| **Leverage** | Cash in hand, immediate pickup, comparable listings, item condition flaws |
| **Key phrases** | "Can you do $X? I can pick up today with cash" |
| **Safety** | Meet in public (police station), bring a friend, inspect thoroughly before paying |
| **Timing** | Items listed 7+ days are ripe for negotiation; sellers get anxious after 2 weeks |

### Bulk Purchasing

| Tactic | Details |
|--------|---------|
| **Opening offer** | Request volume pricing tier; start 20–30% below unit retail |
| **Leverage** | Guaranteed repeat business; upfront payment; flexible delivery timeline |
| **Platforms** | [Alibaba](https://www.alibaba.com/) (MOQ negotiation), wholesale distributors, manufacturer direct |
| **Key phrase** | "We plan to order X units/month ongoing — what's your best unit price for a commitment?" |

### Service Contracts (HVAC, Landscaping, Internet, Insurance)

| Tactic | Details |
|--------|---------|
| **Opening offer** | Get 3+ quotes; present lowest to preferred provider |
| **Leverage** | Competitor pricing, multi-year commitment, referral promises, off-season scheduling |
| **Key phrase** | "I got a quote for $X from [competitor]. Can you match or beat that?" |
| **Retention** | For existing services (internet, insurance): call retention department; mention cancellation |
| **Timing** | HVAC in spring (before summer rush); landscaping in winter; insurance at renewal time |

### Real Estate (Buyer)

| Tactic | Details |
|--------|---------|
| **Opening offer** | 5–10% below asking in buyer's market; at asking in seller's market with strategic contingencies |
| **Leverage** | Pre-approval letter, flexible closing date, inspection-found items, comparable sales |
| **Walk-away** | Never exceed 5% above appraised value |
| **Key phrase** | "We can close quickly with minimal contingencies at $X" |

## Negotiation Communication Templates

### Opening Offer (Email/Message)

```
Subject: Inquiry About [Item] — Serious Buyer

Hello,

I'm reaching out as a purchasing agent on behalf of an interested buyer
regarding [item description].

Based on comparable market data:
- [Comparable 1]: $X
- [Comparable 2]: $X
- [Average market price]: $X

We'd like to offer $[AMOUNT] for [item], with [payment terms].

We're ready to proceed quickly upon agreement. Please let me know your
thoughts.

Best regards,
[Agent Name]
```

### Counter-Offer Response

```
Thank you for the counter of $[AMOUNT].

We appreciate the dialogue. After reviewing updated comparables and
considering [specific factor — condition, market trend, inventory age],
we can move to $[NEW AMOUNT], which reflects:
- [Justification 1]
- [Justification 2]

This represents fair market value based on the data. We remain committed
to a smooth, quick transaction.
```
