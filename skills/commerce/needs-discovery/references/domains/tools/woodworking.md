# Woodworking Tools — Constraints

What's unique to woodworking tool purchases. For generic power/battery/safety guidance, see `index.md`.

## Table Saw

| Type | Price | Notes |
|---|---|---|
| Contractor | $500-1500 | Portable, adequate for occasional use |
| Hybrid | $1000-2500 | Cabinet-grade in smaller footprint |
| Cabinet | $2000-5000+ | Heavy, precise, stationary |

- **Fence quality is critical** — Biesemeyer/T-square style locks solid and stays parallel; cheap fences ruin accuracy
- Miter slot: standard vs T-slot (T-slot accepts accessories)
- Dado stack compatibility — check arbor length and throat plate availability

## Miter Saw

- Sliding vs non-sliding: sliding handles wider boards (crosscut capacity 12"+)
- 10" vs 12": 12" cuts wider stock but heavier and more expensive
- Laser/LED shadow line — shadow line is more accurate than laser (no calibration drift)

## Router

| Type | Best For |
|---|---|
| Fixed base | Edge profiling, table mounting |
| Plunge | Inlay, deep cuts, stopped grooves |
| Palm (trim) | Edge work, laminate trimming |

- Collet size: 1/4" only on small routers; 1/2" needed for large bits
- Variable speed — required for large-diameter bits (prevents burning/kickback)

## Planer (Thickness)

- Portable: $300-700, max width 12-13"
- Stationary: $1500+, max width 20"+
- Knives: HSS (cheaper, dulls faster) vs carbide (lasts 3-5x longer, more expensive)

## Jointer

| Size | Price | Notes |
|---|---|---|
| 6" benchtop | $400-800 | Short beds — limited for long boards |
| 8" standalone | $1000-2500 | Longer beds = better for flattening long stock |

- Check fence adjustment — must lock square and stay square
- Table length matters: longer beds flatten longer boards accurately

## Bandsaw

- 14" standard: $500-1500
- Check resaw capacity (height under blade guide) — determines max board thickness for resawing
- Blade width range: 1/8" (curves) to 3/4"+ (resawing)
- Used market excellent value — check tire condition, wheel alignment, blade guide condition

## Dust Collection

Fine wood dust is **carcinogenic** — dust collection is a health requirement, not a luxury.

| Solution | Price | Notes |
|---|---|---|
| Portable 1-2 HP | $200-500 | Single tool, move hose between machines |
| Single-stage 2-5 HP + ducting | $500-2000 | Whole-shop, fixed ducting |
| Cyclone separator add-on | $150-300 | Separates chips before collector, extends filter life |
| Ceiling air filtration | $200-400 | Filters ambient fine dust |

- N95/P100 respirator minimum if no dust collection is installed

## Sharpening

- Water stones: fast cutting, need flattening maintenance
- Diamond plates: cut fast, no flattening, expensive
- Guided systems (Lansky, Tormek): repeatable angles, slower
- **Sharp tools are safer than dull** — dull tools require more force and slip

## Output Format Example

```
TABLE SAW: contractor tier ($800), T-square fence included, dado-compatible
DUST COLLECTION: portable 1.5 HP ($300) — upgrade path to cyclone later
RESPIRATOR: P100 required (carcinogenic dust) — included in budget
USED OPTION: 14" bandsaw, $450 — tires replaced 2023, wheels aligned
```
