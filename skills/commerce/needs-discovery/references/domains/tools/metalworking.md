# Metalworking Tools — Constraints

What's unique to metalworking tool purchases. For generic power/battery/safety guidance, see `index.md`.

## Lathe

| Size | Price | Swing × Centers |
|---|---|---|
| Mini 7x14 | $500-1000 | 7" swing, 14" between centers |
| Mid-size 9x20 | $1000-2000 | 9" swing, 20" between centers |
| Full-size 12x36 | $3000-8000+ | 12" swing, 36" between centers |

- Check swing over bed, distance between centers, thread cutting range
- Used South Bend / Logan lathes good value — check **ways for wear** (galling, scoring), spindle bearings (runout, noise)

## Mill

| Type | Price | Notes |
|---|---|---|
| Benchtop mill/drill | $500-1500 | Limited rigidity, hobbyist |
| Benchtop (RF-45 class) | $1500-3000 | More rigid, dovetail ways |
| Knee mill (Bridgeport-style) | $3000-15000+ used | Industry standard, heavy |

- Check table travel (X/Y/Z), spindle taper (R8 most common), quill vs fixed spindle
- CNC conversion possible on most manual mills — factor retrofit cost ($500-3000)

## Bandsaw (Metal-Cutting)

- Horizontal/vertical: $300-1000
- Blade speed range — metal cutting needs **slower speeds** than wood (check variable speed or pulley range)
- Coolant system — extends blade life, improves cut quality
- Bi-metal blades for hard metals; carbon blades for soft metals only

## Grinder

| Type | Price | Use |
|---|---|---|
| Bench grinder 6-8" | $50-150 | Sharpening, deburring, light stock removal |
| Belt grinder 2x72 | $500-2000 | Knife making, heavy stock removal |

- Check wheel/belt speed — too fast burns tool steel
- Tool rest quality — rigid, adjustable, close to wheel
- Eye shield and spark guard — must be present and intact

## Measuring Tools

| Tool | Price | Notes |
|---|---|---|
| Calipers (digital) | $30-200 | Mitutoyo is gold standard |
| Calipers (dial) | $40-150 | No battery, durable |
| Calipers (vernier) | $20-80 | No battery, harder to read |
| Micrometers | $50-300 | Higher precision than calipers |
| Machinist square | $20-100 | Check against standard |
| Dial indicator | $30-150 | Setup, tramming, inspection |
| Surface plate (granite) | $50-300 | Precision ground reference |

## Workholding

- Vise: 4-6" machinist vise $50-300 — **Kurt is the standard**
- Clamps, parallels, soft jaws — needed immediately, budget separately
- Lathe chucks: 3-jaw self-centering $100-500 (quick setup); 4-jaw independent $100-400 (precise, slower)

## Safety

- Metal chips are **sharp and hot** — face shield OVER safety glasses, no loose clothing, tie back hair
- Grinding produces sparks — keep flammables away (35 ft clearance)
- Metal dust can be toxic — check if alloy contains **lead, beryllium, or chromium** before machining/grinding dry

## Output Format Example

```
LATHE: mid-size 9x20 ($1400) — 9" swing, 20" centers, threads 8-56 TPI
MEASURING: Mitutoyo digital calipers ($130) + 0-1" micrometer ($60)
WORKHOLDING: 5" Kurt vise ($280) + parallels set ($30)
SAFETY: face shield + N95 (aluminum dust) — not included
USED OPTION: South Bend 10K, $900 — ways good, spindle bearings quiet
```
