# Tools — Generic Constraints

Guidance for helping a user buy tools. This file covers constraints common to ALL tool categories. For category-specific guidance, see the sub-domain table below.

## Power Source

| Source | Pros | Cons | Best For |
|---|---|---|---|
| Corded (120V/240V) | Unlimited power, cheaper upfront, no runtime limit, lighter tool | Cord management, needs outlet, limited mobility | Stationary tools, long sessions, shop work |
| Cordless (battery) | Convenient, portable, no cord snag | Battery ecosystem cost, runtime limits, heavier, battery degradation | Job site work, occasional use, remote locations |
| Pneumatic (air) | Powerful, lightweight tool, no motor heat | Needs compressor + hose, noisy compressor, maintenance | Nailers, impact tools, automotive, high-torque work |

## Battery Ecosystem

When buying cordless tools, you are buying into a **brand platform** — batteries are the biggest long-term cost.

- Buy into ONE brand so batteries interchange across tools.
- Check tool availability within the platform before committing.
- Major platforms:
  - **DeWalt 20V MAX** — broadest line, strong contractor adoption
  - **Milwaukee M18** — premium, pro-focused, extensive specialty tools
  - **Makita 18V LXT** — large line, strong on woodworking
  - **Ryobi 18V ONE+** — budget-friendly, huge variety, lighter duty
- Batteries degrade even in storage — factor replacement cost (3-5 yr).

## Quality Tiers

| Tier | Use Case | Warranty | Examples |
|---|---|---|---|
| Consumer | Occasional DIY, homeowner | 1-3 yr | Ryobi, Black+Decker, entry Craftsman |
| Prosumer / Trade | Daily use, job site | 3-5 yr | DeWalt, Makita, Milwaukee mid-line |
| Industrial | Continuous production | Lifetime (some hand tools) | Snap-on, Klein (electrical), Husky, Craftsman hand |

## Safety Requirements

PPE varies per tool — check what each tool requires before recommending.

- **Eye protection** — safety glasses minimum; face shield for grinding/chipping
- **Hearing protection** — earmuffs or plugs for power tools, compressors
- **Dust mask / respirator** — N95 minimum for dust; P100 for fine/silica dust
- **Gloves** — cut-resistant for sharp stock; NOT for rotating tools (entanglement risk)

## Used Tool Market

| Category | Used Value | What to Check |
|---|---|---|
| Stationary power tools | Excellent | Motor bearings (noise/heat), switch condition, blade/belt wear, table flatness |
| Hand tools | Excellent — nearly indestructible | Rust, handle integrity, edge condition (resharpenable) |
| Battery tools | Fair — risky | Battery age/health (batteries degrade in storage), charger included, tool condition |

