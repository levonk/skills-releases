# Gardening Tools — Constraints

What's unique to gardening tool purchases. For generic power/battery/safety guidance, see `index.md`.

## Hand Tools

| Tool | Price | Notes |
|---|---|---|
| Trowel | $10-30 | Forged single-piece, NOT riveted — rivets fail |
| Transplanting trowel | $10-25 | Depth markings for consistent planting |
| Hand fork/cultivator | $10-25 | Forged tines, not stamped |
| Weeder (fishtail/L-shaped) | $10-20 | Taproot extraction |
| Bypass pruners | $20-60 | Felco #2 is gold standard ($55), replaceable blade |
| Anvil pruners | $15-40 | Dead wood ONLY — crushes live wood |

## Long-Handle Tools

| Tool | Price | Notes |
|---|---|---|
| Round-point shovel | $25-60 | Fiberglass handle or hickory — avoid cheap hardwood |
| Spade (square) | $25-60 | Edging, slicing roots, not for digging |
| Garden fork | $30-70 | Turning soil — NOT for digging rocks (bends tines) |
| Hoe (scuffle/warren/garden) | $20-50 | Style depends on task |
| Bow rake | $20-40 | Soil leveling; leaf rake separate for cleanup |
| Mattock/pickaxe | $30-70 | Hard ground, clay, root removal |

## Pruning Tools

- Loppers: 18-32", bypass for live wood, anvil for dead
- Pruning saw: folding $20-50, fixed blade $30-80 — tri-edge teeth cut on pull stroke
- Pole pruner: 6-12 ft, rope-operated cutter + saw head — check rope condition (used)

## Soil Prep

| Tool | Price | Notes |
|---|---|---|
| Broadfork | $80-200 | Aerates without inverting soil (no-till method) |
| Wheel hoe | $100-300 | Cultivator attachments, manual power |
| Front-tine tiller | $200-500 | Harder to control, walks forward |
| Rear-tine tiller | $500-1500 | Self-propelled, better for hard soil |

- Tine direction: forward-tine vs counter-rotating — counter-rotating bites harder in compacted soil

## Ergonomics

- Tool weight — lighter for repeated use reduces fatigue
- Handle grip diameter: 1.25-1.5" optimal for most hands
- Long handles reduce back strain — match handle length to user height
- Ergonomic grips for arthritis / limited hand strength (ratcheting pruners, cushioned grips)

## Material Quality

| Material | Pros | Cons |
|---|---|---|
| Stainless steel | Rust-resistant, low maintenance | More expensive, slightly softer edge |
| Carbon steel | Holds edge better, cheaper | Needs oiling to prevent rust |
| Aluminum | Light, rust-proof | Bends under load — light duty only |

- Look for **forged, not stamped** heads — forged is stronger, holds shape under stress

## Used Market

| Category | Value | Check |
|---|---|---|
| Hand tools | Excellent — nearly indestructible | Rust (surface OK, pitting bad), edge condition |
| Long-handle | Good | Handle cracks — replace handle $10-20 |
| Power tillers | Fair | Engine compression, tine condition, carburetor |

## Output Format Example

```
PRUNERS: Felco #2 bypass ($55) — replaceable blade, lifetime tool
SHOVEL: round-point, fiberglass handle ($45) — lighter than hickory

ERGONOMICS: long-handle fork (user is 6'1") — reduces back strain

```
