# Appliance Buying Constraints — Overview

> Generic constraints that apply to **all** major appliances. Load the
> sub-domain file for appliance-specific guidance.

## Energy Efficiency

| Signal | Why it matters |
|--------|----------------|
| Energy Star certified | Meets EPA efficiency threshold — ~10-30% less energy than standard |
| EnergyGuide label (yellow) | Estimated annual energy cost + kWh/therms — compare directly across models |
| CEE Tier (Consortium for Energy Efficiency) | Tier I/II/III — higher tier = utility rebates more likely |
| Actual vs rated | Real-world usage varies; rated is a benchmark, not a guarantee |

- Always quote **estimated annual energy cost** from the EnergyGuide label.
- Check for **utility rebates** (often $50-300 for qualified appliances).

## Sizing — Measure the Space

Before recommending any appliance, confirm the user has measured:

- **Width, depth, height** of the cutout / closet / alcove.
- **Door swing clearance** — add 90-130° arc for hinged doors; drawer fronts
  need full extension clearance.
- **Ventilation gaps** — most appliances need 1-2 inches on sides/back/top.
- **Path to the install location** — stairwells, doorways, tight corners can
  block delivery of large units.
- **Floor load** — relevant for laundry (vibration) and full-size refrigerators.

> If the user has not measured, ask before proceeding to model selection.

## Delivery & Installation

| Item | Ask explicitly |
|------|----------------|
| Delivery | Included in price, or extra ($50-200)? Threshold vs in-room? |
| Installation | Basic hookup included, or billed separately ($100-500)? |
| Haul-away of old unit | Included, or $25-75 add-on? |
| Gas line hookup | Requires licensed tech — often extra ($100-300) |
| Electrical (240V / dedicated circuit) | May need electrician — $150-600 |
| Water line (fridge ice, washer) | Braided line + shutoff valve — $50-150 |
| Venting (dryer, range hood) | Rigid duct run, exterior cap — $100-400 |

## Warranty

- **Manufacturer standard**: typically 1-year parts & labor.
- **Extended warranty**: buy from the **manufacturer**, not a third-party
  retailer plan (third-party plans have claim caps, repair limits, and
  exclusions).
- For refrigeration and HVAC, the **sealed system** (compressor, refrigerant
  lines) often carries 5-7 year coverage — this is the warranty that matters,
  not the 1-year general coverage.

## Reliability & Brand Data

- Prefer **Consumer Reports** repair-frequency data and brand reliability
  charts over marketing claims.
- Check **brand-specific known issues** (e.g., certain LG compressors, Whirlpool
  washer bearings) via forums and service-tech channels.
- Ask the user if they have a **preferred brand** and whether local authorized
  service exists for it.

| Dishwasher | 9-12 yr |

| Central AC / Furnace | 15-20 yr |
| Heat pump | 10-15 yr |
| Water heater (tank) | 10-15 yr |
| Water heater (tankless) | 20+ yr |
| Microwave | 7-10 yr |

## Output Format Example

```
Appliance: 24" built-in dishwasher
Energy: Energy Star, ~$35/yr estimated (EnergyGuide)
Sizing: 23.9" W x 24.5" D x 33.5" H — confirm 24" cutout + 1/2" side clearance
Delivery/Install: includes basic hookup + haul-away; hardwire or cord not included
Warranty: 1 yr parts/labor; sealed system n/a
Reliability: Brand X — below-average repair rate per CR
Lifespan: 9-12 yr expected
```
