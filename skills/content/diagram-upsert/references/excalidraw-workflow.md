## Summary (for quick orientation only)

- Excalidraw stores diagrams as `.excalidraw` JSON files (diffable but noisy).
- The hand-drawn aesthetic signals "tentative / WIP" — don't use it where
  precision is required.
- Export to SVG for embedding in markdown; keep the `.excalidraw` source
  alongside the markdown for editability.
- The Obsidian Excalidraw plugin integrates with markdown notes.

The full storage format, export workflow, and integration guidance are in the
bundle page — read it.

## Why this is a pointer

The bundle is the single source of truth. Restating the rules here would
cause drift. When the bundle updates, this pointer stays valid.
