# Prompt-Specific Search

When researching before creating or improving a prompt:

1. **Local**: Scan `internal-docs/prompts/` for existing prompts. Check
   frontmatter and content for similar purpose.
2. **Templates**: Check `config/ai/templates/` for reusable prompt patterns
   (coding, analysis, research) that the new prompt could be based on.
3. **skills.sh / GitHub**: Search for "prompt" + task keywords. Prompt
   patterns are commonly shared.
