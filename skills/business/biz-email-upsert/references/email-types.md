# Email Type Templates

Templates and worked examples for each business email type. Use these as
starting structures when drafting — adapt the content to the specific situation.

## Action Needed

Request someone to do something.

### Structure

```
Subject: Action needed [deadline]: [what needs doing]

[What you need the recipient to do, in one sentence.]

[Two to three lines of context: why it matters, what's blocked, what depends on it.]

[Specific action requested + deadline.]
```

### Example

```
Subject: Action needed today: approve revised offer

Please approve the revised offer for Client X so we can send it by 5 pm.

Legal flagged two clauses that need adjustment (sections 3.2 and 4.1). The
revised draft is attached. Client X expects the offer by tomorrow morning.

Reply with "approved" or requested changes by 3 pm today.
```

## Decision Needed

Request a choice between options.

### Structure

```
Subject: Decision needed [deadline]: [what needs deciding]

[The decision that needs to be made, in one sentence.]

[Two to five lines: the options with facts, names, numbers, dates.]

[Your recommendation and why.]

[Who decides + deadline.]
```

### Example

```
Subject: Decision needed: vendor delay on onboarding

Vendor X missed the March 12 deadline. If we keep the current scope, onboarding
moves to March 19.

Options:
1. Keep full scope, launch March 19 (one week delay)
2. Remove feature Y, launch March 15 (on time, reduced scope)

My recommendation is option 2 — feature Y is nice-to-have and the March 15
date was committed to the client.

Please reply by 3 pm today so we can confirm with the client.
```

## Update

Inform stakeholders of status.

### Structure

```
Subject: Update: [what's updated]

[The headline: what changed, in one sentence.]

[Two to five lines: details with names, numbers, dates.]

[Next milestone or next update timing.]
```

### Example

```
Subject: Update: contract signed with Acme

Acme signed the contract today. The engagement starts March 1 with a 6-month
initial term, value $180K.

Kickoff meeting is scheduled for March 3 at 10 am. I'll send the agenda by
Friday.

No action needed — this is for your awareness.
```

## Risk

Flag a problem or potential issue.

### Structure

```
Subject: Risk: [what's at risk]

[The risk, in one sentence. Bad news first.]

[Two to five lines: what happened, impact, likelihood.]

[Recommendation or mitigation + owner + deadline.]
```

### Example

```
Subject: Risk: campaign under target this week

The campaign is 18 percent below target after four days.

Likely cause is narrow audience targeting. At current rate, we'll miss the
monthly target by 12 percent.

Recommendation: expand targeting today and review performance again tomorrow
at 2 pm. I'll own the adjustment and send an update by end of day.
```

## Escalation

Raise an issue to a higher authority.

### Structure

```
Subject: Escalation: [what's escalated]

[The issue and why it's escalated, in one sentence.]

[Two to five lines: what was tried, what failed, what's blocked.]

[What you need from this person + deadline.]
```

### Example

```
Subject: Escalation: engineering resource conflict on Project X

Project X is blocked because the two assigned engineers were pulled to
incident response on Project Y. I've raised this with the Y lead but no
resolution yet.

Project X's deadline is March 15. At current staffing, we'll miss it by 5
days. I need either the engineers back by Monday or approval to contract a
temporary replacement.

Can you decide by end of day Friday so we can adjust the plan?
```

## Confirmation

Verify an agreement or understanding.

### Structure

```
Subject: Confirmation: [what's confirmed]

[What was agreed, in one sentence.]

[Two to three lines: specifics — who, what, when, where.]

[Ask for confirmation or corrections.]
```

### Example

```
Subject: Confirmation: Q2 budget allocation

Following our meeting today, confirming the Q2 budget allocation:

- Engineering: $120K (3 hires)
- Marketing: $45K (campaign + tools)
- Operations: $30K (infra + vendor renewals)

Please reply with "confirmed" or corrections by Wednesday so I can finalize
the plan.
```
