# prompt-upsert

## Summary

Creates new AI prompts, modifies and improves existing prompts, and
scaffolds reusable prompt files with companion READMEs. Transforms user
input into precision-crafted, structured prompts using the Levonk
methodology.

## Version

1.0.0

## Methodology

DECONSTRUCT → DIAGNOSE → DEVELOP → DELIVER

1. **DECONSTRUCT** — Break down the vague request into components
2. **DIAGNOSE** — Identify what makes the request unclear or incomplete
3. **DEVELOP** — Build the structured prompt with thinking triggers and validation hooks
4. **DELIVER** — Produce the final prompt file with companion README

## Modes

- **Mode A: Create** — Create a new prompt from scratch
- **Mode C: Update** — Update an existing prompt (upsert, with audit checklist)

## Key Capabilities

- Transform vague requests into structured reusable prompts
- Add thinking triggers and validation hooks
- Align prompts with reusable templates
- Scaffold multi-prompt task sets with parallel/sequential numbering
