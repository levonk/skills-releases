# Build System Essentials

A compounding knowledge base documenting practices for build system
orchestration — Makefile structure, standard target conventions, and the
principles that keep build automation maintainable across projects. Each
concept captures the patterns that promote readability, consistency, and
modular builds.

## Concepts

* [Overview](overview.md) - Synthesis of build system principles and how Make, Just, and orchestration fit together
* [Makefile Essentials](makefile-essentials.md) - Core Makefile guidelines: centralized scripting, standardized targets, documentation-driven help, and modular structure
* [Build Target Conventions](build-target-conventions.md) - Mandatory, preferred, and potential Makefile targets (clean, build, test, lint, deploy, etc.) with their responsibilities
