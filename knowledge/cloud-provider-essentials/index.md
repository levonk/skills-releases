# Cloud Provider Essentials

This knowledge bundle documents best practices for major cloud providers —
AWS, Azure, GCP, and OCI — including infrastructure patterns, cost
optimization strategies, security guidance, networking configurations, and
Infrastructure as Code (IaC) implementations using Pulumi and Terraform.

| [GCP Essentials](gcp-essentials.md) | GCP | Cloud Storage, Cloud SQL, Firestore, VPC, Compute Engine best practices |
| [OCI Essentials](oci-essentials.md) | OCI | Object Storage, Autonomous Database, VCN, compute, and security best practices |
